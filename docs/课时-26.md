## 课时 26：插件市场

现有插件

### webpack-box

webpack-box 服务。基础配置，这个包是必须要安装的。

### @pkb/plugin-react

可以解析 jsx 文件，使用 react

### @pkb/plugin-eslint

eslint 插件，可以执行 `webpack-box eslint`

### @pkb/plugin-stylelint

stylelint 插件，可以执行 `webpack-box stylelint`

### @pkb/plugin-tslint

tslint 插件，可以执行 `webpack-box tslint`
